# Computer Networks – Client Server Project

## Description
This project implements a simple TCP client-server communication in Python.
The server listens on port 6000 and handles up to 5 clients sequentially.

## Files
- server.py – TCP server implementation
- client.py – TCP client implementation
- part2_tcp_port6000.pcapng – Wireshark capture of the TCP communication

## How to Run
1. Run server.py
2. Run client.py multiple times (5 clients)
3. Capture traffic using Wireshark on loopback interface
